# hexo-theme-butterfly

Butterfly 是一款美观且功能丰富的 Hexo 主题，专为博客和内容展示设计。它结合现代网页设计趋势，提供流畅的用户体验和强大的自定义功能，适合技术博客、个人站点和企业站点使用。

## 🚀 快速开始

### 💾 安装方式

#### 方法一：Git 安装（推荐）

1. 确保你的系统已安装 Git。
2. 在 Hexo 项目根目录下运行以下命令：

```bash
git clone https://github.com/jerryc127/hexo-theme-butterfly.git themes/butterfly
```

3. 修改 Hexo 配置文件 `_config.yml` 中的 theme 字段为 `butterfly`。

#### 安装稳定版本（推荐）

使用 Hexo 官方推荐的方式安装：

```bash
npm install hexo-theme-butterfly --save
```

#### 安装开发版本（抢先体验新功能）

如果你希望尝试最新的功能，可以安装开发分支：

```bash
git clone -b dev https://github.com/jerryc127/hexo-theme-butterfly.git themes/butterfly
```

#### 方法二：NPM 安装

```bash
npm install hexo-theme-butterfly --save
```

## ⚙️ 主题配置

1. 复制主题配置文件 `_config.yml` 到你的 Hexo 根目录。
2. 修改配置文件中的各项参数以适配你的需求，例如站点信息、菜单、颜色主题等。
3. 启动 Hexo 服务查看效果：

```bash
hexo clean && hexo g && hexo s
```

## ✨ 主题特色

### 🎨 设计风格

- **响应式布局**：支持各种设备访问，包括手机、平板和桌面。
- **暗色模式（Dark Mode）**：夜间阅读更舒适。
- **阅读模式（Read Mode）**：提供沉浸式阅读体验。
- **自定义主题色**：支持多种颜色风格切换。

### 📝 内容功能

- 支持文章摘要、标签、分类、时间线等。
- 内置文章卡片、友链页面、说说动态等功能模块。
- 可自定义首页、归档、标签、分类等页面。

### 🔍 搜索与导航

- 集成 Algolia、本地搜索等多种搜索方案。
- 支持全站搜索和关键词高亮。

### 🎨 代码展示

- 内置 Prism.js 和 Highlight.js 支持代码高亮。
- 支持 Mermaid 图表、Katex 数学公式、Markdown 渲染等。

### 💬 社交互动

- 支持 Disqus、Valine、Waline、Gitalk、Utterances 等多种评论系统。
- 可集成社交媒体链接、分享按钮等。

### 📊 数据分析

- 支持 Google Analytics、Umami、百度统计等分析工具。

### 🎪 视觉效果

- 内置粒子动画、加载动画、返回顶部按钮等增强用户体验。
- 支持 PWA（渐进式 Web 应用）功能。

### 🛠️ 进阶功能

- 支持第三方插件集成（如音乐播放器 APlayer、数学公式渲染、图表库等）。
- 提供多种小工具（Widget）展示最新文章、热门标签、分类归档等。
