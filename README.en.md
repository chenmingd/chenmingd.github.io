# hexo-theme-butterfly

Butterfly is a beautiful and feature-rich Hexo theme designed for blogs and content presentation. It incorporates modern web design trends to deliver a smooth user experience and powerful customization options, making it ideal for technical blogs, personal sites, and corporate websites.
